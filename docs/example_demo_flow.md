# Example Demo Flow

1. Run the app with `python run_web.py`.
2. Select `rule_based` or configure OpenRouter/OpenAI in the sidebar.
3. Upload a messy CSV/XLSX file.
4. Ask: `apa yang bisa dilakukan dari data ini?`
5. Ask: `buat warna kolom divisi jadi merah`
6. Ask: `buat tabel terpisah berdasarkan divisi`
7. Confirm the pending split-sheet plan.
8. Ask: `buat summary total sales per region`
9. Ask: `highlight baris yang statusnya overdue`
10. Ask: `export workbook`
11. Download the resulting workbook from the sidebar or output section.
